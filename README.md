# Бот-градусник для чего-то

Бот градусник (очень полезное)

---

## Быстрый старт

### 1. Предварительные требования

- [Git](https://git-scm.com/) ≥ 2.34

### 2. Клонирование репозитория

   ```bash
      git clone https://github.com/whyfourwhentwo-sketch/practice-maxbot.git
      cd practice-maxbo
   ```
### 3. Установка зависимостей

   ```bash
      poetry install
   ```

### 4. Настройка переменных окружения

1. Переименуйте скопированный файл:

```bash
   mv .env.example .env
   ```

2. Отредактируйте .env при необходимости


### 5. Запуск приложения

   ```bash
      poetry run python main.py
   ```
