# Описание 

Интерактивный дашборд для мониторинга и визуализации проблем в сфере ЖКХ. Данные можно отфильтровать по ID чата, платформе(Telegram/MAX) и временному периоду

## Используемые технологии

`React` - библиотека для построения пользовательских интерфейсов\
`Chart.js 4` - ,иблиотека для создания графиков и диаграмм\
`react-chartjs-2` - обертка для интеграции Chart.js с React

## Структура проекта
```
frontend_charts/
├── src/
│   ├── components/
│   │   ├── Dashboard.js      # Основной дашборд (используется)
│   │   ├── Dashboard.css     # Стили для дашборда
│   │   ├── MyChart.js        # Тестовый график (не используется)
│   │   └── PieChart.js       # Тестовый круговой график (не используется)
│   ├── App.js                # Главный компонент (подключает Dashboard)
│   ├── App.css               # Глобальные стили
│   └── main.js               # Точка входа
├── public/
│   └── index.html
├── package.json
└── README.md
```
